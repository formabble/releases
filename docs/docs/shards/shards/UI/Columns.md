| `Output ➡️` ||The output of this shard will be its input. | | [`Any`](../../types/#any) |
| `Contents` | :fontawesome-solid-circle-plus:{title="No"} No  | A sequence of UI contents. | `[]` | [`[None Shard [Shard]]`](../../types/#seq) |
